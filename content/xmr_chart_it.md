## XmR Chart-It (Web App)

The XmR Chart-It web application is a free to use, XmR chart generating tool. User can generate an XmR chart from a list of existing datasets or upload their own dataset (CSV). For those that are unfamiliar with the tool, the app serves as a practical introduction to XmR charts. For those familiar with the charts it serves as a free alternative to pay-to-play software like Minitab and JMP.
